# 🧍 PostureCorrector

## 📌 Overview
PostureCorrector is an AI-based application that helps users maintain correct body posture. It uses computer vision techniques to detect body alignment and provides real-time feedback to improve posture habits.

## 🚀 Features
- Real-time posture detection
- Alerts for incorrect posture
- Easy-to-use interface
- Works with webcam
